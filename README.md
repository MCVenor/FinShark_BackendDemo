## C# Backend Demo Store

### Status: 🏗️ Im Aufbau (Lernprojekt)

Dieses Repository dient als technische Demo, um moderne 
Software-Architektur mit .NET zu explorieren. 
Der Fokus liegt nicht auf maximaler Performance, sondern auf Clean Code, 
der Anwendung von Entwurfsmustern (Design Patterns) 
und einer klaren Trennung von Verantwortlichkeiten.
